Volver al cuerpo

Créditos

Propósito

Este es un proyecto que busca la práctica de etiquetas de HTML y aplicar estilos de CSS a todo el sitio web. La idea es desarrollar más los conocimientos y afianzarlos, para que, a través de la práctica, no se olviden y tengamos éxito en el mundo empresarial.

Forma de uso

Consiste en que al presionar una parte del cuerpo redirige a una página de esos músculos, en esa página la primera imagen tiene enlaces dentro de la misma página que mueven la vista a la información del músculo. Hasta ahora está hecho para computador, arrastra el cursor sobre los músculos y allí te dará el nombre con el enlace para llevarte a la información.

Cómo lo logré

Estuve usando Visual Studio como framework. En su gran mayoría es HTML, está dividido en una página principal y 3 que contienen la información del cuerpo humano.
Hay 4 archivos de CSS, uno para cada página, ya que quise experimentarlo así, me parece una forma mejor de organizar todo el trabajo, ya que al abrir cada archivo veo una parte específica y no tengo que recordar dónde están los elementos de tal página entre columnas interminables de código CSS.
Usé Paint.Net para crear el logo, el cual es una imagen de perfil (como las de Google) usando gafas oscuras.

Nota

Ningún material de imágenes me pertenece, solo lo que respecta a HTML y CSS.
